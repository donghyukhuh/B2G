# 맞춤형 나라장터 입찰정보 대시보드

GitHub Pages에 바로 게시 가능한 단일 페이지 대시보드입니다.

## 포함 기능
- 3개 영역(사전규격 / 입찰공고 / 가격입찰(낙찰하한율)) 분리 조회
- 기본 키워드 10개 자동 세팅 및 키워드 추가/삭제
- 참가가능지역 필터: 서울, 경기, 전국만 허용
- 각 영역 최대 10건 표시
- 표 항목: 공고명, 금액, 공고기관, 입찰마감일시, 파일첨부문서 링크
- 업데이트 기준: 한국시간 오전 8시 자동 갱신 + 수동 새로고침 버튼

## 배포
`main` 브랜치에 푸시하면 GitHub Pages에서 `index.html`이 바로 반영됩니다.
